
# A股均线测算工具

一个纯前端、零依赖的 A 股均线测算工具。输入 6 位股票代码，即可获取最近 35 个交易日的 K 线数据，并计算 MA5/MA10/MA20 均线在 **-10% ~ +10%** 涨跌幅区间下的预测收盘价与均线值，辅助短线交易决策。

## 功能特性

- **均线测算** — 基于最近 35 个交易日收盘价，计算 MA5 / MA10 / MA20
- **涨跌幅预测** — 覆盖 -10% 至 +10%（含），每 1% 一档，显示各档对应的预测收盘价及均线值
- **K 线图表** — 原生 Canvas 绘制 K 线图 + 三条均线，支持暗色/亮色主题切换
- **数据源** — 调用腾讯股票 API（`web.ifzq.gtimg.cn`），无需 API Key
- **本地缓存** — 查询历史自动存入 `localStorage`，下次输入时提供自动补全
- **一键复制** — 可将测算结果以文本格式复制到剪贴板
- **移动端适配** — 响应式布局，手机浏览器可直接使用
- **零依赖** — 单个 `.html` 文件，双击即可运行，无需安装任何软件

## 快速开始

1. 打开 [`stock-ma-tool.html`](stock-ma-tool.html)，用浏览器直接打开
2. 在输入框中输入 6 位股票代码（如 `600519`）
3. 点击 **查询** 按钮
4. 查看均线测算表格与 K 线图

> **支持板块：** 沪市（6/9 开头）、深市（0/3/4/8 开头）

## 界面说明

| 区域 | 说明 |
|------|------|
| 输入区 | 输入 6 位股票代码，支持历史自动补全 |
| 状态栏 | 显示股票名称、最新价、市场状态（盘中/已收盘）、当前 MA 值 |
| 涨跌幅表格 | 左列为涨幅（0~+10%），右列为跌幅（0~-10%），点击行可高亮选中 |
| K 线图 | 实时 K 线 + 均线走势，虚线右侧为预测区域 |
| 复制按钮 | 将测算结果导出为纯文本 |

## 技术细节

- **数据接口** — 腾讯 K 线 API：`web.ifzq.gtimg.cn/appstock/app/fqkline/get`
- **均线算法** — 简单移动平均（SMA），取最近 N 日收盘价均值
- **预测逻辑** — 将涨跌幅作用于最新价作为第 36 日收盘价，重新计算 MA5/10/20
- **图表渲染** — 原生 HTML5 Canvas，无第三方图表库
- **主题切换** — CSS 变量 + `localStorage` 持久化
